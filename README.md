## 🔗 API Endpoints

### Публічні

**GET /api/data**
Получити всі дані для форми (ароматы, кольори, мило)

**POST /api/order**
Створити нове замовлення

### Адміністративні

**POST /api/admin/login**
Відданення в адмін-панель

**GET /api/admin/orders**
Получити всі замовлення (потребує auth)

**DELETE /api/admin/orders/:id**
Видалити замовлення (потребує auth)

**POST /api/admin/scents**
Додати аромат (потребує auth)

**DELETE /api/admin/scents/:id**
Видалити аромат (потребує auth)

**POST /api/admin/colors**
Додати колір (потребує auth)

**DELETE /api/admin/colors/:id**
Видалити колір (потребує auth)

**POST /api/admin/soaps**
Додати тип мила (потребує auth)

**DELETE /api/admin/soaps/:id**
Видалити тип мила (потребує auth)

**POST /api/admin/contacts**
Додати контакт (потребує auth)

**DELETE /api/admin/contacts/:id**
Видалити контакт (потребує auth)
